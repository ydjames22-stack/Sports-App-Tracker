# Campus Training Planner App

## 1. Project Description
The Campus Training Planner App is a mobile application designed to help student athletes organize their training schedules and workouts. The app allows users to plan practices, track workouts, and manage daily training tasks using their phone. The goal of the app is to help athletes stay organized and improve their time management.

## 2. Problem Being Addressed
Many student athletes struggle with managing practice schedules, workouts, and other responsibilities. Training plans are often written down or spread across multiple apps, which makes them difficult to track. This app will solve that problem by providing one place where athletes can organize their workouts and schedules.

## 3. Platform
The app will be developed for the Android mobile platform. It will be created using MIT App Inventor. The application will run on Android phones and tablets.

## 4. Front-End and Back-End Support

### Front-End
The front-end is the part of the app that users interact with. This includes buttons, menus, and screens where users can enter workout information or view schedules.

### Back-End
The back-end will store and manage the data entered into the app. This may include workout logs, schedules, and user information. Data may be stored using TinyDB or another simple storage option.

## 5. Functionality
The main features of the application will include:
- Creating a training schedule
- Logging workouts and exercises
- Viewing daily or weekly training plans
- Tracking workout progress
- Saving and updating workout data

## 6. Design (Wireframes)
The app will include several simple screens:

- Main Screen: Navigation to workouts, schedules, and progress
- Workout Screen: Enter exercises, sets, reps
- Schedule Screen: View daily/weekly plan
- Progress Screen: Track workout history

## 7. GitHub Documentation
GitHub will be used to document and organize the project. The repository will include the project outline, updates, and version control. The README file contains the project description and development details.
